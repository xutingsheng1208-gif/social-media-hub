## 技术架构亮点

### 后端架构
```
backend/
├── src/
│   ├── controllers/    # RESTful API控制器
│   ├── services/       # 业务逻辑层
│   ├── extractors/     # 平台内容提取器
│   ├── routes/         # 路由定义
│   └── models/         # 数据模型
├── prisma/             # 数据库schema
└── uploads/            # 媒体文件存储
```

### 前端架构
```
frontend/
├── src/
│   ├── components/     # 通用UI组件
│   ├── pages/         # 页面组件
│   ├── stores/        # Zustand状态管理
│   ├── services/      # API服务层
│   ├── types/         # TypeScript类型定义
│   └── hooks/         # 自定义Hooks
```

### 数据库设计
- **categories**: 支持自引用的二级分类
- **contents**: 内容主表，支持多种媒体类型
- **tags**: 标签表
- **content_tags**: 内容标签关联表

## 安装和运行

### 环境要求
- Node.js 16.0+
- npm 或 yarn
- 至少 2GB 磁盘空间

### 快速启动
```bash
# 1. 后端设置
cd backend
npm install
npx prisma generate
npx prisma migrate dev
npm run dev

# 2. 前端设置
cd frontend
npm install
npm start
```

### 访问地址
- 前端应用: http://localhost:3000
- 后端API: http://localhost:3001

## 项目价值

这个项目提供了一个完整的个人内容管理解决方案，特别适合需要收集和整理社交媒体内容的用户。通过自动化的内容提取和强大的搜索管理功能，用户可以轻松建立自己的数字资源库。

项目采用现代化的技术栈，具有良好的扩展性和维护性，可以作为学习全栈开发的优秀案例。
